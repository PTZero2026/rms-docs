# Mô hình dữ liệu

Thực thể cốt lõi (chi tiết bổ sung dần theo feature):

- **DeTai** — thông tin đề tài, trạng thái vòng đời.
- **DotKeuGoi** — kỳ nhận đề xuất.
- **NguoiDung / VaiTro** — tài khoản và phân quyền.
- **HoiDong / PhieuCham** — đánh giá, nghiệm thu.
- **BaoCaoTienDo**, **GiaoDichKinhPhi**, **SanPhamKhoaHoc**.

> Nhúng ERD bằng Mermaid `erDiagram` tại đây.
