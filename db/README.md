# db/ — Lược đồ CSDL

| Thư mục | Nội dung |
|---|---|
| `migrations/` | Migration tăng dần, version hóa. Phản ánh [data-model.md](../docs/architecture/data-model.md). |
| `seed/` | Dữ liệu khởi tạo (danh mục B01, vai trò/quyền B03…). |

CSDL: PostgreSQL. Lưu UTC; định dạng hiển thị xử lý ở tầng app.
